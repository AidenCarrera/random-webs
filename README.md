# Random Webs

A collection of interactive web experiments built with **Next.js**, **React**, and **Tailwind CSS**.

## Purpose

To explore creative coding concepts, unique UI/UX interactions, and audio synthesis through a series of "mini-websites".

## Features

- A collection of small interactive tools and experiments.
- A beat-maker drum sequencer powered by Tone.js.
- Visual effects demos.
- Simple utility pages like timers and tests.e.

## Tech Stack

- **Next.js 16** – Frontend framework
- **React 19** – UI library
- **Tailwind CSS v4** – Styling
- **Tone.js** – Audio synthesis
- **Framer Motion** – Animations

## Installation and setup

To clone and run this application:

```bash
# Clone this repository
git clone https://github.com/AidenCarrera/random-webs.git

# Go into the repository
cd random-webs

# Install dependencies
pnpm install

# Run the development server
pnpm run dev
```

Open http://localhost:3000 to view the website.
